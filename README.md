# Telebot Command Center

This is a Telegram bot command center project.

## Features
- Manage bot commands easily
- Run on localhost:3123
- Frontend dashboard

## How to Run
1. Install dependencies: `npm install`
2. Start the server: `npm run dev` or via PM2
3. Open browser at `http://localhost:3123`

## Project Structure
- `/src` - source code
- `/dist` - build output
- `/node_modules` - dependencies
